# 🌐 Multilingual AI Translator

A powerful AI-based translator supporting Urdu (script + Romanized), English, and more — built with **Gradio + Groq (LLaMA 3.3)**.

---

## 🚀 Live Demo

👉 **[Click here for Live Demo](https://sameenfatimaa-urdu-translator.hf.space)**

---

## ✨ Features

* 🔄 Bidirectional translation (8 languages)
* 🔡 Romanized Urdu detection
* 🎚️ Tone control (Casual / Neutral / Formal)
* 🔊 Text-to-Speech
* 📂 File upload (.txt)
* ⬇️ Download translations
* 📊 Analytics dashboard

---

## 🛠️ Tech Stack

* Python
* Gradio 6.x
* Groq API (LLaMA 3.3)
* Hugging Face Spaces

---

## 📦 Installation (Local)

```bash
pip install -r requirements.txt
python app.py
```

---

## 🌍 Deployment

Deployed on Hugging Face Spaces
👉 https://huggingface.co/spaces/Sameenfatimaa/urdu-translator

---

## 🔐 Note

Make sure to add your `GROQ_API_KEY` in environment variables.
